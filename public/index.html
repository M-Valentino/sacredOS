Sacred OS can boot in three ways: from IndexedDB, from a JSON disk backup, or with a fresh install. If this is your first time using Sacred OS, you will want to boot with a fresh install.

Your data is automatically saved to IndexedDB, which allows you to store gigabytes of data. You can boot from IndexedDB to resume where you left off.

If you frequently clear your browser's data or want to use Sacred on a different device, You want to boot from a JSON disk backup. JSON disk backups will save your session exactly as how IndexedDB does, but you are able to save it as a file anywhere on your real computer. You can create these backups in the files.html program.
